# 技术写作者工具箱

写文档总从空白页开始？
**模板+变量替换，秒出标准化文档。**

[![HTML](https://img.shields.io/badge/打开即用-无需安装-orange)]()

## 文档不再"从头写"

- **变量替换**：name/version/author三字段自动替换
- **模板收藏**：最多30个模板，localStorage存储
- **模板生成**：根据配置自动生成技术文档模板

## 怎么用

打开 `template_generator.html` → 选择模板 → 填写变量 → 复制
