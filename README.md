# 技术写作者工具箱

> 技术文档写作的模板库、检查清单与最佳实践

---

## Features / 功能特点

| 功能 | 说明 |
|------|------|
| API文档模板 | REST/GraphQL 接口文档模板，含参数表+示例+错误码 |
| 快速入门模板 | 新用户上手指南模板，5步以内完成目标 |
| 故障排查模板 | 技术支持文档模板，症状→原因→解决方案 |
| 发布说明模板 | 版本更新日志模板，新功能+修复+兼容性 |
| 架构说明模板 | 系统设计文档模板，组件说明+数据流 |
| 中英双语示例 | 每个模板均提供中英双语示例 |
| 可填充使用 | 标准文档结构，直接替换内容即可 |
| 符合规范 | 遵循 Google 技术文档规范标准 |

## Installation / 安装

```bash
# 克隆仓库
git clone https://github.com/yourusername/technical-writing-toolkit.git

cd technical-writing-toolkit

# 查看模板目录
ls 写作模板/
```

## Usage / 使用方法

### 基础用法

直接进入 `写作模板/` 目录，选择需要的模板复制使用。

```bash
# 查看可用模板
ls 写作模板/

# 阅读模板
cat 写作模板/API文档模板.md
```

### 模板类型概览

| 模板 | 文件 | 适用场景 | 特点 |
|------|------|----------|------|
| API 文档 | `API文档模板.md` | REST/GraphQL 接口文档 | 参数表 + 示例 + 错误码 |
| 快速入门 | `快速入门模板.md` | 新用户上手指南 | 5步以内完成目标 |
| 故障排查 | `故障排查模板.md` | 技术支持文档 | 症状→原因→解决方案 |
| 发布说明 | `版本发布说明模板.md` | 版本更新日志 | 新功能 + 修复 + 兼容性 |
| 架构说明 | `架构说明模板.md` | 系统设计文档 | 图 + 组件说明 + 数据流 |

### API 文档模板示例

```markdown
# API 名称

## 接口说明
[简要说明接口功能]

## 请求地址
`GET /api/v1/{resource}`

## 请求参数

| 参数名 | 类型 | 是否必填 | 说明 |
|--------|------|:--------:|------|
| id | string | 是 | 资源ID |
| page | integer | 否 | 页码，默认1 |

## 响应示例

```json
{
  "code": 0,
  "data": {
    "id": "123",
    "name": "示例"
  },
  "message": "success"
}
```

## 错误码

| 错误码 | 说明 | 处理方式 |
|:------:|------|----------|
| 400 | 参数错误 | 检查请求参数 |
| 404 | 资源不存在 | 确认资源ID |
```
