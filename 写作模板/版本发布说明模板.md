# 版本发布说明模板

## 版本号

```
# vX.Y.Z - YYYY-MM-DD
```

## 模板结构

```markdown
# v1.0.0 - 2026-05-30

## 新功能
- [功能1]：[描述]
- [功能2]：[描述]

## 修复
- [修复1]：[描述]
- [修复2]：[描述]

## 兼容性
- [兼容性说明]

## 升级指南
1. [步骤1]
2. [步骤2]
```

## 示例

```markdown
# v2.1.0 - 2026-03-15

## 新功能
- 新增用户权限管理模块
- 支持 OAuth2.0 第三方登录
- 优化搜索性能，响应时间降低 50%

## 修复
- 修复了文件上传时的内存泄漏问题
- 修复了深色模式下的文字显示异常

## 兼容性
- 保持 v2.0 API 接口完全兼容
- 数据库结构无变更，可直接升级

## 升级指南
1. 备份当前数据库
2. 下载新版本包
3. 覆盖旧版本文件
4. 运行升级脚本: `python upgrade.py`
```
